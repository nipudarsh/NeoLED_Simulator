# NeoLED Arduino Simulator

A **web-based Arduino LED simulator** that allows you to visually program LEDs using Arduino-style code. This project focuses on **neon LED effects** and is perfect for experimenting with Arduino LED circuits entirely in your browser.

- Simulate **digitalWrite(), pinMode(), analogWrite(), delay()**
- **Run, Pause, Reset** buttons with live console output
- Demo templates: Blink LED, Knight Rider, Police Siren
- Works fully in the browser (offline capable)
